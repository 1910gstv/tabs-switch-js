# Tabs Switch - JS

### ❓ O que é?

Esse aplicativo é um selecionador de guias, ou seja, de acordo com a guia selecionada, ela irá exibir um conteudo diferente da outra.
<br>
Usamos HTML, CSS e Javascript para a construção dos códigos.
